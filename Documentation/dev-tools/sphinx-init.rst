===========
Sphinx Init
===========

Symlinking this file is okay.

This repo must exist at same level as your repo.

Your cwd must be the root of your repo.

SYNOPSIS
========

::

	sphinx-init.sh

DESCRIPTION
===========

This script creates a directory named 'Documentation' on your repo, and
symlinks Sphinx config files along with a Makefile from this repo into that
directory.
