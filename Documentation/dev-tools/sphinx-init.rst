===========
Sphinx Init
===========

Core library must exist at same level as your repo.

Your cwd must be repo root.

SYNOPSIS
========

::

	sphinx-init.sh

DESCRIPTION
===========

This script creates a dir named 'Documentation', and symlinks Sphinx config
files along with a Makefile into that dir.
