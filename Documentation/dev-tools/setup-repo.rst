================
Setup Repository
================

Symlinking this file is okay.

This repo must exist at same level as your repo.

Your cwd must be the root of your repo.

SYNOPSIS
========

::

	setup-repo.sh

DESCRIPTION
===========

This script sets up following things for your repo:

   - shows a pretty commit message before you write your crap
   - nags you about typos before letting you commit
