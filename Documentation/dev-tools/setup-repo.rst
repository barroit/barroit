================
Setup Repository
================

Core library must exist at same level as your repo.

Your cwd must be repo root.

SYNOPSIS
========

::

	setup-repo.sh

DESCRIPTION
===========

This script sets up following things for your repo:

   - shows a pretty commit message before you write your crap
   - nags you about typos before letting you commit
