======================
The Core Terminologies
======================

The core uses some special terminologies. You must know these, or you'll get
confused in some sentences.

``this repo``, ``this/repo``

	Refers to ``barroit`` repository, where this document lives.

``your repo``, ``your/repo``

	Refers to another repository, often the target repository.

``your script``, ``your/script``

	Refers to another script, often the target script.

``this setup``
	Refers to barroit's setup.

``your email``, ``your_email``
	Refers to email address of the target user.
