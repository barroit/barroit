======================
The Core Terminologies
======================

The core uses some special terminologies. You must know these, or you'll get
confused in some sentences.

1.	this repo, this/repo

		Refer to ``barroit`` repository, where this document lives.

2.	your repo, your/repo

		Refer to another repository, often the target repository.
